# JoltPhysics.js — Интерактивная симуляция мягкого тела

Интерактивная 3D-демонстрация физики мягких тел с использованием JoltPhysics.js и Three.js.

## ✨ Особенности

- 🧵 **Мягкое тело** — ткань 40×40 вершин с реалистичной физикой
- 🎯 **6 типов объектов** — шары, кубы, капсулы, цилиндры, конусы, торы
- 📱 **Мобильное управление** — перетаскивание пальцем, адаптивный интерфейс
- 🖱️ **Перетаскивание ткани** — тяните полотно мышью или пальцем
- ⚡ **До 500+ объектов** — отличная производительность
- 🎨 **Красивая визуализация** — тени, освещение, туман
- 📊 **Реальный индикатор загрузки** — отслеживание прогресса 0→100%

## 🎮 Управление

| Действие | Способ |
|----------|--------|
| Перетащить ткань | Зажать ЛКМ/палец на ткани и двигать |
| Вращать камеру | Зажать ПКМ или двумя пальцами |
| Добавить объект | Нажать кнопку с типом объекта |
| Толчок объектов | Кнопка "💨 Толчок" |
| Очистить всё | Кнопка "🗑️ Очистить" |

## 🛠️ Технологии

- **JoltPhysics.js** — физический движок
- **Three.js** — 3D рендеринг
- **WebAssembly** — высокая производительность
- **ES Modules** — современная архитектура
